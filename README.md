# PHP Evolution video made with Gource

Files and settings to produce a PHP source code evolution from SVN and Git repositories using [Gource](https://code.google.com/p/gource/).

## Usage

```
$ ./build
```

## Requirements

* Gource 0.43
* ffmpeg
